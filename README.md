# Pixel Playground – Community Gaming Website (PHP)

**Pixel Playground** adalah website komunitas game yang dirancang untuk meningkatkan interaksi antar gamer. Website ini menyediakan ruang bagi pengguna untuk berbagi review game, berdiskusi, berbagi tips & trik, serta mencari teman bermain.

---

## Tujuan Pengembangan
- **Membangun komunitas gamer** melalui forum dan ruang interaksi online.
- **Menyediakan informasi dan review game** dari sesama pengguna.
- **Meningkatkan pengalaman bermain**, dengan mempermudah gamer menemukan partner untuk bermain bersama.
- Menjadi tempat berbagi **tips, trik, strategi**, dan insight lain terkait dunia game.
- Menjadi platform yang dapat dimanfaatkan **developer game** untuk mendapatkan feedback dan mempromosikan game mereka.

---

## Fitur Utama
- **Login & Register** – autentikasi pengguna.
- **User Profile** – informasi profil dan aktivitas pengguna.
- **Forum Diskusi** – ruang bertukar pendapat, tanya jawab, dan berbagi pengalaman.
- **Game Review** – fitur untuk menulis, membaca, dan membagikan ulasan game.
- **Main Bareng** – mencari teman untuk bermain game secara online.
- Navigasi halaman yang modular dan mudah dipahami.

---

## Teknologi & Tools
- **PHP** – backend processing & page rendering.
- **HTML, CSS, JavaScript** – tampilan dan interaksi frontend.
- **XAMPP / Localhost** – server development.
- **Canva** – pembuatan desain UI dan konsep tampilan awal (prototype).
